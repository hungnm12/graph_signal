# SEIR-Campus Repo Evaluation Note

## Evaluation Setup

This evaluation uses the public sample dataset from the original `SEIR-Campus` repository. The `SEIR-Campus` simulation is executed directly from the cloned GitHub repo, and the same dataset is converted into an aggregated weighted graph for the `Weighted SIR` baseline.

To keep the experiment computationally feasible, the run uses a sampled subnetwork of `246` students selected by enrollment activity from the original dataset.

The pseudo-ground-truth epidemic curve is generated by `SEIR-Campus` with `rate=0.00003630` and no spontaneous community exposure.

## Metrics

- `MSE` and `MAE` compare the infected curve against the pseudo-ground-truth SEIR-Campus curve.
- `Curve Accuracy = 1 - MAPE` is used as an accuracy-style fit metric.
- `R^2` reports how much infected-curve variation is explained by each method.

## Result Summary

`SEIR-Campus` achieved `MSE=1.191`, `MAE=0.890`, `Curve Accuracy=0.674`, and `R^2=-0.200`.
`Weighted SIR` achieved `MSE=352.190`, `MAE=10.865`, `Curve Accuracy=0.000`, and `R^2=-353.760`.

Under this dataset-driven evaluation, `SEIR-Campus` fits the SEIR-Campus outbreak trajectory better. This is consistent with the fact that SEIR-Campus explicitly models the exposed stage and day-by-day contact structure.

## Reporting Note

These values should be presented as outbreak-curve fitting metrics, not classifier accuracy, because both methods are epidemic simulators.
